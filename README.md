# Battleship-Game
This is a simple implementation of the classic Battleship game using JavaScript. The game consists of a 12x12 grid where players can place their ships and take turns attacking each other's ships. The first player to sink all of the opponent's ships wins the game.

## Features
- Two players can play against each other.
- Players take turns attacking each other's grid.
- The game keeps track of hits and misses.
- The game ends when one player sinks all of the opponent's ships.
